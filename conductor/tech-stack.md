# Tech Stack

## 1. Frontend
*   **Language:** JavaScript (Vanilla ES6+)
*   **3D Engine:** Three.js (via CDN)
*   **Physics Engine:** Rapier (via CDN)
*   **Styling:** CSS3 (Standard)

## 2. Infrastructure & Tools
*   **Development Server:** Simple local server (e.g., `python3 -m http.server` or `live-server`).
*   **Build Process:** No build step (Direct use of ESM in the browser).
*   **Version Control:** Git

## 3. Libraries & Dependencies
*   **Three.js:** Used for 3D rendering, scene management, and camera controls (OrbitControls).
*   **Rapier:** Used for real-time physics simulation of falling hearts and collisions.
